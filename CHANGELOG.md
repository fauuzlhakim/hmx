# Changelog

## 0.1.0 - 2026-04-08

Initial public-ready release.

Highlights:
- split the CLI into focused `hmxlib` modules
- added packaging via `pyproject.toml` and `setup.py`
- added `hmx` console entry point
- added GitHub Actions CI for tests and compile checks
- made tests portable for clean-clone verification
- added public documentation, contributing guide, and MIT license
- improved default path portability with home-directory and environment-variable based runtime paths
- added `--version`
